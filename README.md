# Informatics 2 Course Assignments

Welcome to my repository! This project contains my coursework, assignments, and implementations for my university's Algorithms and Data Structures course. 

## About this Repository

All assignments and algorithmic problem-solving exercises in this repository are implemented in **C**. The theoretical foundation for these implementations is based on the course literature:

> **Introduction to Algorithms, Fourth Edition**
> by Thomas H. Cormen, Charles E. Leiserson, Ronald L. Rivest, and Clifford Stein (CLRS)
> *MIT Press, 2022*

## Course Scope / Topics Covered

1. Introduction, basic sorting, recursion
2. Complexity and correctness
3. Divide and conquer, recurrences
4. Heap sort, quick sort
5. Pointers, lists, sets, abstract data types
6. Trees, AVL trees
7. Hash tables
8. Dynamic programming
9. Graph algorithms
10. Minimum Spanning Trees, Shortest Paths

## Tech Stack

* **Language:** C
* **Tools:** Standard C compiler, Git
